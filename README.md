# Simple CRUD API

Simple API to try some libs and learn.

### Requirements

- Python 3.8 or superior
- Docker

### Run

To run the API with the postgres database we just need to build and up the docker compose:

```docker-compose up --build```

### API documentation

It's automatically generated by swagger and with docker compose up locally 
it can be accessed at http://localhost:8000/docs

### PG Admin
Can be accessed at http://localhost:5050 with the data used on docker-compose file.
